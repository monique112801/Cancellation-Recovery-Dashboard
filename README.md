# Patient Cancellation Recovery Analysis

## Overview
This project analyzes how effectively a clinic fills last-minute patient cancellations and identifies inefficiencies in outreach and scheduling processes. The analysis focuses on patient response behavior, therapist performance, and contact effort required to recover appointments.

## Tools Used
- Tableau
- Excel

## Skills Demonstrated
- Data Cleaning and Preparation  
- KPI Development  
- Data Visualization and Dashboard Design  
- Trend Analysis  
- Process and Workflow Optimization  

## Key Insights
- Fill rate was 25%, indicating limited success in recovering cancelled appointments  
- A large portion of outreach attempts resulted in "No Answer," reducing booking effectiveness  
- Multiple contact attempts (2–3 on average) were often required to secure a booking  
- Fill rates varied across therapists and time slots, suggesting differences in scheduling efficiency
